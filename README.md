# Cloud Computing - CI/CD Assignment 2

## Project Overview
This project is a practical application for the Cloud Computing course. It demonstrates how to set up a **CI/CD Pipeline** using **GitHub Actions** to automatically deploy a static website to **GitHub Pages**.
